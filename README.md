# SpaceWarp++

SpaceWarp++ is a full remake of [SpaceWarp](https://github.com/Wam25/SpaceWarp) in [Rust](https://www.rust-lang.org/)

### Planned Features:
 - Level editor
 - Better Jumping
 - Better levels
 - Better art *(created by [txtur](https://github.com/txtur))*
 - Texture packs
 - Actual Puzzles
